### üîß Installation Issues

#### Common Issues and Solutions

**Docker services not starting:**
```bash
# Check Docker status
docker-compose ps

# Check logs
docker-compose logs [service-name]

# Restart services
docker-compose restart
```

**Python import errors:**
```bash
# Reinstall package
poetry install

# Check Python path
python -c "import sys; print(sys.path)"
```

**Memory issues:**
```bash
# Check memory usage
free -h

# Check ZRAM status
zramctl
```
